# 📚 مكتبة إلكترونية

HTML و CSS و Typerscript  تطبيق ويب بسيط لإدارة مكتبة إلكترونية باستخدام 
يوفر إمكانية عرض الكتب، إضافتها، حذفها، والبحث عنها مع فلترة حسب النوع.

##  التقنيات المستخدمة

- إضافة كتاب جديد

-  البحث عن كتاب
-  فلترة حسب النوع
